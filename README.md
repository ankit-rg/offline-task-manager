# 📑 Personal Web Dashboard

A lightweight, offline-capable personal dashboard built entirely with vanilla web technologies. It serves as a unified hub for managing bookmarks and tracking daily tasks, utilizing browser `localStorage` to keep your data private and instantly accessible.

## ✨ Features

**🔖 Bookmark Management**
* Organize links into custom categories with emoji support.
* 15+ color themes for visual category distinction.
* One-click link copying and quick-launch arrows.
* Real-time search filtering.

**✅ Task Tracker**
* Create, edit, and delete tasks with priority levels (Low, Medium, High).
* Set due dates and track overdue tasks automatically.
* Filter by priority or custom category.
* **Browser Notifications:** Get reminded of pending tasks every 5 minutes (requires browser permission).
* Swipe animations for completing and restoring tasks.

## 🛠️ Tech Stack
* **HTML5 & CSS3:** Responsive grid layout, custom animations, and variables.
* **Vanilla JavaScript:** No frameworks or build tools required.
* **Local Storage:** Zero-backend architecture. Data is stored directly in your browser.

## 🚀 How to Use
1. Clone this repository or download the ZIP file.
2. Open the `index.html` file directly in any modern web browser.
3. Start adding your bookmarks and tasks! 

*Note: Since the app relies on `localStorage`, your data will remain on the specific browser and device you use to open the file.*
